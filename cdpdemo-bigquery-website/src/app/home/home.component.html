Guide

Retail CDP Demo with Lytics & Tamr


Tutorial
  • Step 1 : Create a new user account in simulated ecommerce platform
  • Step 2 : Check out any product using fictitious data
  • Step 3 : Request support through live chat
  • Step 4 : Create audience in lytics (New User + Requested Support)
  • Step 5 : Activate (Send Voucher to Email)

Process
  • User registration and shopping activites stream to BigQuery using Google Analytics
  • All datasets in BigQuery including from other ecommerce will be unified using TAMR
  • New audience / segmentation is created using LYTICS
  • Email campaign assign to selected audience using MailChimp
