Guide

Retail CDP Demo with Lytics & Tamr


Tutorial
  1. Create a new user account in simulated ecommerce platform
  2. Check out any product using fictitious data
  3. Request support through live chat
  4. Create audience in lytics (New User + Requested Support)
  5. Activate (Send Voucher to Email)

Use case 1
  • User A has registered and purchased product from ecommerce Site A
  • User A visited ecommerce Site B for the first time
  • With identity resolution , User A as higher likelihood to make a purchase

Process
  1. User registration and shopping activites stream to BigQuery using Google Analytics
  2. All datasets in BigQuery including from other ecommerce will be unified using TAMR
  3. New audience / segmentation is created using LYTICS
  4. Email campaign assign to selected audience using MailChimp
